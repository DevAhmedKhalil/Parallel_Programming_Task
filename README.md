# Parallel Programming Task: User Registration and Login

This repository contains a simple C program for user registration and login. The program is designed to be run in a Code::Blocks IDE environment.

## Program Overview

This C program demonstrates a basic user registration and login system. It allows users to:

- Register with a username, email, and password.
- Log in using their registered username and password.

The program uses simple data structures to store user registration information and checks for valid login credentials. It serves as a basic example for educational purposes.
